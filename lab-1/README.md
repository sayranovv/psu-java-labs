Пример 2:<br />
a=2 b=15 num=33<br />

### Алгоритм решения

```java

    int max = Math.max(a, b);
    
    return num >= min && num <= max;
}
```

Пример 1:<br />

Пример 2:<br />

результат: “44 года”

### Алгоритм решения

```java
public String age(int x) {
    int lastDigit = x % 10;
    int lastTwoDigits = x % 100;

    if (lastDigit == 1 && lastTwoDigits != 11) {
        return x + " год";
    } else if ((lastDigit == 2 || lastDigit == 3 || lastDigit == 4) && (lastTwoDigits != 12 && lastTwoDigits != 13 && lastTwoDigits != 14)) {
        return x + " года";
    } else {
        return x + " лет";
    }
}
```

Подсказки:<br />

intу=123/10; // у будет иметь значение 12

Пример 1:<br />

результат: false

### Алгоритм решения

```java
public boolean equalNum(int x) {
    if (x < 0) x = -x;
    if (x == 0) return true;
    
    int first = x % 10;
    x /= 10;
    
    while (x > 0) {
        if (x % 10 != first) {
            return false;
        }
        
        x /= 10;
    }
    
    return true;
}
```

выведите на экран количество попыток, которое потребовалось пользователю,
чтобы угадать число.

Пример:<br />
Введите число от 0 до 9:<br />
5<br />
Вы не угадали, введите число от 0 до 9:<br />
9<br />
Вы угадали!<br />
Вы отгадали число за 2 попытки

### Алгоритм решения

```java
public void guessGame() {
    Random random = new Random();
    int secret = random.nextInt(10);
    
    Scanner scanner = new Scanner(System.in);
    int attempts = 0;
    int guess;
    
    do {
        System.out.println("Введите число от 0 до 9:");
        guess = scanner.nextInt();
        attempts++;
        
        if (guess != secret) {
            System.out.println("Вы не угадали, ");
        }
    } while (guess != secret);
    
    System.out.println("Вы угадали!");
    System.out.println("Вы отгадали число за " + attempts + " попытки");
}
```

```java
public void reverse(int[] arr) {
    for (int i = 0; i < arr.length / 2; i++) {
        int temp = arr[i];
        arr[i] = arr[arr.length - 1 - i];
        arr[arr.length - 1 - i] = temp;
    }
}
```

```java
    int[] result = new int[arr1.length + arr2.length];
    
    System.arraycopy(arr1, 0, result, 0, arr1.length);
    System.arraycopy(arr2, 0, result, arr1.length, arr2.length);
    
    return result;
}
```

### Алгоритм решения

```java
public int[] findAll(int[] arr, int x) {
    int count = 0;
    
    for (int num : arr) {
        if (num == x) {
            count++;
        }
    }
    
    int[] indices = new int[count];
    int index = 0;
    
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == x) {
            indices[index++] = i;
        }
    }
    
    return indices;
}
```

```java
    for (int num : arr) {

        }

    int[] result = new int[count];

    for (int num : arr) {
```
